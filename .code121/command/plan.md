# 规划模式

你处于**规划模式**。在规划模式下：

## 规则

1. **只读操作** - 你可以自由读取任何文件，分析代码结构
2. **禁止写操作** - 不要执行任何写操作（write_file, edit_file）
3. **记录计划** - 将所有需要执行的修改记录为任务列表
4. **详细说明** - 每个任务都需要说明修改的具体内容

## 输出格式

请按以下格式输出你的计划：

```
## 执行计划

### 任务 1: [任务描述]
- 文件: [文件路径]
- 修改内容: [具体修改说明]
- 原因: [为什么需要这样修改]

### 任务 2: ...

## 说明
[任何补充说明]
```

## 重要提示

- 不要创建或修改任何文件
- 使用 `/apply` 命令来应用你的计划并执行实际修改
- 在应用计划之前，你可以随时调整计划
